BLOCK 3 – COMMUNICATION PRACTICE

One challenge I faced while learning coding was implementation.
I was able to understand the logic of the problem, but I did not know how to write the code for it. 
I got stuck while converting my idea into a program. This made me feel confused and slow.
To overcome this, I started practicing regularly. I solved simple problems and focused on writing code step by step.
I also checked solutions and understood how others implemented the same logic. Slowly, I improved my coding skills.
From this, I learned that understanding logic is not enough, we must practice coding daily to improve implementation.
