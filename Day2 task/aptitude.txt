BLOCK 1 – APTITUDE (Profit & Loss)

Question 1
A shopkeeper buys a book for ₹400 and sells it for ₹480.
Find the profit percentage.  
 Answer:
       
       Cp= 400
       sp=480
       profit=?
       profit=sp-Cp
       pr=480-400
       pr=80
       profit %= profit/cp *100
               = 80/400 *100
       profit %= 20% 

Question 2
A man buys a watch for ₹1500 and sells it for ₹1200.
Find the loss percentage.
  Answer:
       
       Cp = 1500
       sp= 1200
       loss= ?
       loss = cp-sp
            = 1500 - 1200
            =300
       loss % = loss/cp *100
            = 300/1500 *100
       loss % =20%

Question 4
If the cost price of 5 pens is ₹100, what is the selling price of each pen to make 25% profit?
  Answer:
     
        cp of 5 pens=100

        cp of 1 pen =100/5
                    =20

           profit=25% of 20

BLOCK 3 – COMMUNICATION PRACTICE 

My final year project title is “Canteen Pre-Order System”.The main problem we wanted to solve is that students and faculty are usually stand in long queues in the canteen, which wastes time, especially during breaks to buy their snacks and foods.
So, we developed an website where users can pre-order their food ,snacks and pick it up without waiting.
For this project, we used MongoDB, Express.js, React, and Node.js (MERN stack) and Jwt Authentication. We also included features like user login, menu display, order placement, and order tracking.
My role in the project was as a frontend developer. I worked on both frontend. I created the UI using React and handled database.
One of the main challenges we faced was handling real-time order updates and managing multiple users at the same time. Initially, there were some issues with data syncing and payment, but we fixed them through testing and debugging.
Overall, this project improved my technical skills and teamwork experience.
